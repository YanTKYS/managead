# v0.9.7 受入テスト結果記録

## 実施目的

v0.9.7 は、v1.0.0 本番初版へ進む前に、実運用担当者の観点で DirectoryReadOnly による実 AD 参照、画面表示、ヘルプ、配布物の分かりやすさを確認するための受入テスト版です。

## 実施バージョン

| 項目 | 内容 |
|---|---|
| 対象バージョン | v0.9.7 受入テスト版 |
| 起動モード | DirectoryReadOnly |
| 位置づけ | v1.0.0 前の受入テスト |
| 前提 | v0.9.6 開発側打鍵テスト反映済み |

## 確認済み項目

| 項目 | 結果 | 備考 |
|---|---|---|
| 実 AD ユーザー検索 | 確認済み | DirectoryReadOnly で検索できることを確認 |
| ユーザー属性表示 | 確認済み | 主要属性を画面で確認できる |
| 所属グループ表示 | 確認済み | ユーザーの所属グループを確認できる |
| コンピュータ参照 | 確認済み | コンピュータ検索・参照ができる |
| グループ参照 | 大きな問題なし | グループ検索・メンバー表示の大きな問題は確認されていない |
| 利用者向けヘルプ | 追加済み | `help/index.html` を同梱 |
| release-build | 整備済み | 手動実行でリリース ZIP を作成する運用 |

## 指摘事項

v1.0.0 リリースを妨げる重大な指摘は確認されていません。

| 区分 | 内容 | 対応 |
|---|---|---|
| A | v1.0.0 前に必ず対応が必要な重大問題 | なし |
| B | v0.9.7 内で可能なら対応する軽微な改善 | ヘルプ追加・説明整理を実施済み |
| C | v1.0.0 以降に回す改善 | `docs/roadmap.md` / `docs/backlog.md` で管理 |

## v1.0.0 前に対応した事項

- v0.9.6 開発側打鍵テストで出た起動、表示、グループ参照、ログ確認、配布物整理の指摘を反映済み。
- v0.9.7 受入テストで指摘された「ドキュメントは充実しているが、テキストとして視認性が悪い」点に対し、利用者向けローカル HTML ヘルプを追加済み。
- release-build で `help/` を成果物に含める構成を整備済み。
- v1.0.0 リリースノートを追加済み。

## v1.0.0 以降に回す事項

- エラーメッセージの改善
- ヘルプファイルの更新
- 表示項目の微調整
- 配布手順の改善
- 設定ミス時の案内強化
- グループメンバー表示の改善
- 検索条件の追加
- 表示項目の設定化
- CSV 出力の拡張
- エラー診断ガイド、FAQ、設定チェック機能

AD 更新機能、グループ追加・削除、ユーザー無効化、退職処理、OU 移動、GPO 編集、一括更新処理は、参照専用ツールとして安全に展開する方針と合わないため、対象外または慎重検討として扱います。

## v1.0.0 リリース判断

v0.9.7 受入テストで重大な問題は確認されていないため、ManageAdTool は **v1.0.0 本番初版としてリリース可** と判断します。
