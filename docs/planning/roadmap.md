# ManageAdTool ロードマップ（保守用メモ）

v1.0.0 以降の正式なロードマップは `../roadmap.md` を参照してください。

## 現在の方針

ManageAdTool は閉域ネットワーク向けの **Active Directory 参照専用確認支援ツール** として運用します。

- 実 AD に対する操作は `DirectoryReadOnly` による参照のみです。
- AD 更新機能、グループ追加・削除、ユーザー無効化、退職処理、OU 移動、GPO 編集、一括更新処理は実装しません。
- v1.0.0 後の改善は、安定化、参照機能強化、運用支援、ヘルプ・手順書改善を中心に扱います。
