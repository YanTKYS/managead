# v0.9.6 開発側打鍵テスト記録（要約）

詳細な記録は `docs/operation/test-record-v0.9.6.md` を参照してください。

## 位置づけ

v0.9.6 は、開発側打鍵テストで出た起動、表示、グループ参照、ログ確認、配布物整理などの指摘を反映した版です。

## v1.0.0 リリース判断との関係

v0.9.6 時点の未確認事項や追加確認事項は、v0.9.7 受入テストで確認しました。v0.9.7 受入テストで重大な問題が確認されなかったため、v0.9.6 の記録は v1.0.0 本番初版へ進む判断と矛盾しません。

## 主な確認・反映事項

- 起動時 ServiceMode 選択とメイン画面遷移の改善
- DirectoryReadOnly でのユーザー検索・属性表示・所属グループ表示
- コンピュータ参照
- グループ参照・メンバー表示改善
- 検索 UI の不要項目削除
- ログ確認画面の表示改善
- self-contained 単一 exe リリースパッケージ整理

## 後続確認
